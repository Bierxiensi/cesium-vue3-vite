# Vue 3 + Vite

项目基于 vue3 + vite + cesium，已实现常见三维动画场景，欢迎有兴趣的同学加入共建，一起搞开源，一起学习，一起乐呵呵~~

## 项目启动

```
npm i
npm run dev
```
